# Constructor for bcea

Constructor for bcea

## Usage

``` r
new_bcea(df_ce, k)
```

## Arguments

- df_ce:

  Dataframe of all simulation eff and cost

- k:

  Vector of willingness to pay values

## Value

List object of class bcea.
