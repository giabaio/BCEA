# Set Reference Group

Alternative way to define (e,c) reference group.

## Usage

``` r
setReferenceGroup(he) <- value

# S3 method for class 'bcea'
setReferenceGroup(he) <- value

# Default S3 method
setReferenceGroup(he) <- value
```

## Arguments

- he:

  A `bcea` object containing the results of the Bayesian modelling and
  the economic evaluation.

- value:

  Reference group number

## Value

bcea-type object
