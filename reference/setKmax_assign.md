# Set Maximum Willingness to Pay

Alternative way to define `K` statistic.

## Usage

``` r
setKmax(he) <- value

# S3 method for class 'bcea'
setKmax(he) <- value

# Default S3 method
setKmax(he) <- value
```

## Arguments

- he:

  A `bcea` object containing the results of the Bayesian modelling and
  the economic evaluation.

- value:

  Maximum willingness to pay

## Value

bcea-type object
