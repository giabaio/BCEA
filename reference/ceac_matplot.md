# CEAC Matrix Plot

CEAC plot using `matplot` in Base R.

## Usage

``` r
ceac_matplot(he, pos_legend, graph_params, ceac)
```

## Arguments

- he:

  A `bcea` object containing the results of the Bayesian modelling and
  the economic evaluation.

- pos_legend:

  Legend position

- graph_params:

  Aesthetic ggplot parameters

- ceac:

  `ceac` index in `he`
