# EIB parameters specific to base R plot

EIB parameters specific to base R plot

## Usage

``` r
eib_params_base(he, graph_params, cri_params)
```

## Arguments

- he:

  A `bcea` object containing the results of the Bayesian modelling and
  the economic evaluation.

- graph_params:

  Type of plot device

- cri_params:

  Credible interval parameters

## Value

list
